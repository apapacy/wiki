# Обновление до Ubuntu 18.04
1. Обновление пакетов:
```text
apt-get update && apt-get upgrade && apt-get dist-upgrade
```
2. Удаление мусора:
```text
apt-get autoremove
```
3. Перезагрузка (обязательно):
```text
reboot
```
4. Обновление до Ubuntu 18.04
```text
do-release-upgrade -d
```
